<h1 align="center">Zonn</h1>

<p align="center">

</p>

<p align="center">
  A minimalist focus timer that lives in your menu bar. Start sessions instantly, control Spotify, and stay in flow with a floating timer overlay.
</p>

## Download

Visit **[zonn.app](https://zonn.app)** for download instructions.

---

## Features

- **Menu bar app** — always accessible, never in the way
- **Global keyboard shortcut** (⌘⇧F) to start focus mode
- **Floating timer overlay** that stays on top
- **Spotify integration** — playback controls and now-playing display
- **Session history** and statistics
- **Clean, native macOS design**

## Requirements

- macOS 14.0 (Sonoma) or later
